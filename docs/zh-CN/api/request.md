# 请求头

在发送请求时，需要在header中携带以下参数：

- `appId`: 应用 ID
- `timestamp`: 时间戳，13 位长度
- `sign`: 签名

## 请求头参数

| 参数名      | 描述           | 示例值               |
|-------------|----------------|----------------------|
| appId       | 应用 ID        | your-app-id          |
| timestamp   | 时间戳         | 1699261493465        |
| sign        | 签名           | 生成的签名字符串     |
